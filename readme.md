# Deskripsi

Modul praktikum ini merupakan bagian integral dari mata kuliah **Jaringan Komputer** yang menggunakan textbook **"Computer Networking: A Top-Down Approach" edisi ke-8** karya **J.F. Kurose** dan **K.W. Ross**.

Praktikum ini dirancang untuk memberikan pengalaman **hands-on** dalam memahami protokol jaringan komputer melalui:

- Analisis paket menggunakan **Wireshark**
- Pemrograman **socket**

---

# Daftar Modul

## Modul 1: Running Modul
- Pengenalan aturan dan sistem pelaksanaan praktikum
- Verifikasi tools yang akan digunakan

## Modul 2: Pengenalan Tools
- Instalasi dan konfigurasi Wireshark
- Dasar-dasar packet sniffing
- Analisis paket sederhana

## Modul 3: HTTP
- Basic HTTP GET/response interaction
- HTTP conditional GET
- Retrieving long documents
- HTML documents dengan embedded objects
- HTTP authentication

## Modul 4: DNS
- Penggunaan `nslookup`
- Penggunaan `ipconfig`
- Tracing DNS dengan Wireshark
- Analisis query dan response DNS

## Modul 5: UDP
- Investigasi protokol UDP
- Analisis header UDP
- Port dan payload UDP

## Modul 6: TCP
- Analisis transfer TCP
- Three-way handshake
- TCP congestion control
- Flow control
- RTT dan throughput analysis

## Modul 7: Socket Programming
- Membuat aplikasi jaringan dengan UDP
- Membuat aplikasi jaringan dengan TCP
- Python socket programming

## Modul 8: Pembagian Topik Tugas Besar
- Plotting kelompok tugas besar
- Penentuan topik proyek akhir

## Modul 9: Web Server
- Membuat web server sederhana
- HTTP request/response handling
- TCP socket programming untuk web server

## Modul 10: IP
- Analisis datagram IPv4
- IP fragmentation
- Pengenalan IPv6
- Traceroute analysis

## Modul 11: DHCP
- Dynamic Host Configuration Protocol
- DHCP Discover, Offer, Request, ACK
- IP address assignment

## Modul 12: ICMP dan Asistensi Tugas Besar
- ICMP dan Ping
- ICMP dan Traceroute
- Progress report tugas besar

## Modul 13: Ethernet and ARP
- Analisis frame Ethernet
- Address Resolution Protocol (ARP)
- ARP caching

## Modul 14: 802.11 WiFi
- Wireless networking protocol
- Beacon frames
- Data transfer
- Association / Disassociation

## Modul 15: Presentasi Tugas Besar
- Demo aplikasi
- Presentasi hasil proyek
- Pengumpulan laporan

## Modul 16: Asistensi 2 (Praktikum Susulan)
- Session untuk praktikum susulan
